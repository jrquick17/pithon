## Setup

### Install

#### Mac
`sudo pip install pygments docutils`
`brew install sdl2 sdl2_image sdl2_ttf sdl2_mixer gstreamer`

`pip install cython`
`pip install kivy`

`cd vendor/kivy-md && py setup.py install`

### Run 
    
`python main.py`

### Syntax Highlighting
#### IntelliJ
